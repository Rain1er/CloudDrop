ExecCode方法，学到了一个trick。在PHP中使用输出缓冲。避免直接输出结果，而是先存储在缓冲区中，等待加密再返回。
```php
function main() {
	ob_start(); // 开始输出缓冲
	%s          // 执行传入的代码
	return ob_get_clean(); // 获取缓冲区内容并作为返回值
}
```
